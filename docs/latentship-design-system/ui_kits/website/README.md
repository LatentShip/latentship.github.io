# LatentShip Website UI Kit

A high-fidelity interactive recreation of the LatentShip marketing site.

## Design Width
1140px container, full-page layout.

## Pages
- **Home** — Hero, proof strip, Why Us, Roadmap, Featured Projects, FAQ, Footer
- **Our Work** — Filterable project grid
- **About Us** — Founder bios
- **Get Started** — Multi-step intake form (3 steps)
- **Project Detail** — Individual case study view

## Components
| File | Contents |
|---|---|
| `Nav.jsx` | Pill navbar, brand mark SVG (rocket), animated active link, green CTA |
| `Hero.jsx` | Split hero: copy panel (light) + timeline card (dark panel-dark) |
| `ProjectCard.jsx` | Project card with image/gradient preview, tag chips, duration; also `ProjectGrid` with tag filter |
| `RoadmapSection.jsx` | 3-step roadmap on green stage bg + 3 trust cards |
| `Footer.jsx` | Dark CTA panel + light footer grid |

## Usage
Open `index.html` to see the full interactive prototype. Click nav links and project cards to navigate between pages.
